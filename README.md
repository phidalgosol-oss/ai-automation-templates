# ai-automation-templates

Practical AI automation workflows you can actually use.

No fluff. No theory. Just working templates built with real tools — 
n8n, Zapier, Claude, and WhatsApp — to help professionals automate 
the repetitive parts of their work and focus on what matters.

---

## What's inside

### 🤖 AI workflows
- **Daily briefing bot** — morning summary of your calendar, tasks, 
  and priorities sent to WhatsApp via n8n + Claude
- **Email triage assistant** — AI categorizes and drafts replies for 
  common email types
- **Meeting prep workflow** — auto-generates context doc before any 
  calendar event

### 💬 WhatsApp automations
- **Reminder system** — set reminders via WhatsApp message, 
  get them back at the right time
- **Quick capture** — send a voice note or message, get a structured 
  summary + action items back

### 📋 Prompt templates
- **Weekly review prompt** — structured reflection for professionals
- **Project kickoff prompt** — clarifies scope, blockers, and next 
  steps fast
- **LinkedIn post framework** — turns a raw idea into a structured draft

## How to use these

Each folder contains:
- A plain-language explanation of what the workflow does
- Step-by-step setup instructions
- The n8n JSON or Zapier Zap export (where applicable)
- Notes on customization

No coding required for most workflows.
